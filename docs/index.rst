Extracts and enriches Jira data for Kanban


Installation
------------

To install Chiketto ,
run this command in your terminal:

.. code-block:: console

   $ pip install chiketto


Usage
-----

Chiketto's usage looks like:

.. code-block:: console

   $ chiketto [PROJECT]* [OPTIONS]

.. option:: -f <ISO-8601 date>, --from-date <ISO-8601 date>

    The initial date for pulling data.

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
